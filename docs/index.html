公司简介

公司成立于2016年11月17日，座落于安徽省淮北市相山区相山南路盛世商贸城。 公司注册资本6800万元，拥有员工92人。 公司自成立以来，不断谋求新的发展，承接了多项房屋建筑、市政、装修装饰，电力工程，公路工程，水利工程等工程，积累了丰富的施工经验，掌握了各种新型施工技术。多年来，安徽凤宇筑建筑工程有限公司恪守“质量第一、客户至上”的服务宗旨，遵循“优质、高效、团结、奉献”的工作态度，为社会创造了一批优质精品工程。

公司架构人员简介

公司机构设置总经理技术部、经营部、项目部、财务部（结算中心）4个职能科室， 各类管理人员和技术人员共计92人，其中一级建造师2人，二级建造师14人，高级职称2人，中级职称38人,初级职称22人，技工14人等
